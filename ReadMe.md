# MagiCreator 

MagiCreator is a pip package to create pip style package structure (Yes, self-reference!) targeting self-developed projects.

Usage:
    
```bash
    cd magicreator
    python setup.py sdist    
    pip install . 
    magicreator <package_name>
```
